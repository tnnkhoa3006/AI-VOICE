# MiMo Voice Studio - Agent Skill Pack

Bộ file này dùng để định hướng xây dựng app AI Voice Studio sử dụng MiMo Text-to-Speech.

Mục tiêu sản phẩm:
- Người dùng nhập text/script.
- AI hỗ trợ tối ưu nội dung.
- Người dùng chọn voice/style.
- App generate audio bằng MiMo TTS.
- Lưu project, segment, audio.
- Export WAV/MP3.
- Có credit, billing, lịch sử, voice clone/design về sau.

Cấu trúc file:
- 00_product_flow.md: Flow sản phẩm đầy đủ.
- 01_tech_stack.md: Công nghệ đề xuất.
- 02_architecture.md: Kiến trúc hệ thống.
- 03_database_schema.md: Schema database.
- 04_api_contract.md: API contract.
- 05_frontend_agent.skill.md: Skill agent cho frontend.
- 06_backend_agent.skill.md: Skill agent cho backend.
- 07_audio_worker_agent.skill.md: Skill agent cho worker xử lý audio.
- 08_ai_prompt_agent.skill.md: Skill agent cho AI/script/style prompt.
- 09_security_qa_agent.skill.md: Skill agent bảo mật, QA, test.
- 10_project_rules.md: Rule chung khi phát triển.
- 11_mvp_task_board.md: Task board triển khai MVP.
